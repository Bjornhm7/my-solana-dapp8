# my-solana-dapp8

A Solana Mobile dApp built with Solana Builder.

## Building

This project uses GitHub Actions to automatically build APKs.

### Automatic Builds
- Push to `main` branch triggers a build
- Or manually run from Actions tab

### Download APK
After a successful build, download the APK from the workflow artifacts.

## Development

Built with:
- Kotlin
- Jetpack Compose
- Material 3
